# awesome-openclaw-skills

A collection of community skills for [OpenClaw](https://github.com/openclaw/openclaw) agents.

## Skills

| Skill | Description |
|-------|-------------|
| [systematic-event-analysis](./systematic-event-analysis/) | Framework for analyzing how large-scale events impact investment assets and generating actionable signals. [Live demo](https://iran-war-dashboard.vercel.app) |

## Usage

Copy the skill folder into your OpenClaw workspace:

```bash
cp -r <skill-name> ~/.openclaw/workspace/skills/
```

## Contributing

PRs welcome! Each skill should be a folder containing at minimum a `SKILL.md` file.
